Examples


このサンプルには、CKImagesという画像が含まれています。
この画像はサンプル中のImageページで使用するものです。
通常CGIディレクトリでは画像を表示できないので、適切なディレクトリにコピーして下さい。
コピー先のディレクトリは、リソースディレクトリとしてExample.cgiで設定して下さい。
リソースディレクトリは以下の部分です。

# resource directory
resource = '/images'
